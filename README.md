# Computer Science Capstone Portfolio

## Project Overview

This repository contains my Computer Science Capstone completed at Southern New Hampshire University. Rather than developing a single application, this capstone demonstrates my ability to analyze, enhance, and improve software across three core areas of computer science: software engineering and design, algorithms and data structures, and database systems. The included artifacts showcase practical experience with secure coding, RESTful APIs, MongoDB, software testing, debugging, and technical documentation.

---

## Project Objectives

- Enhance an existing software application
- Improve the user interface and overall user experience
- Implement secure user authentication
- Integrate a MongoDB database
- Develop RESTful APIs
- Validate application functionality through testing
- Demonstrate professional software engineering practices

---

## Key Enhancements

### User Interface Improvements

- Added images and visual enhancements to improve usability and appearance.
- Improved navigation and overall user experience.

### Authentication

- Implemented user login functionality.
- Created secure account registration and password management features.

### RESTful API Development

- Developed RESTful APIs to support communication between the application and the database.
- Tested API endpoints using Postman.

### Database Integration

- Connected the application to MongoDB.
- Implemented CRUD operations for application data.

### Testing & Debugging

- Performed application testing using Postman.
- Diagnosed and resolved application issues throughout development.
- Verified all functionality after resolving integration and configuration issues.
